MUTUAL, CONDITIONAL MUTUAL AND INTERACTION INFORMATION.

The script computes mutual, conditional mutual and interaction information
using the compiled routing by Peng (mi.0.912.zip).

https://www.mathworks.com/matlabcentral/fileexchange/14888-mutual-information-computation

The folder has to be unzipped and added to the working path. 
If the cpp files do not work in your Matlab, recompile the m files.

timeseries.MAT = matrices of simulated timeseries object of the study.
